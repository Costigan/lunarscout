Source file is DSN_horizon_masks.xlsx

A file that begins as follows
10.5
10.6
10.7
10.7
...

Refers to a mask of 10.5 degrees at azimuth angle 0, 10.6 degrees at angle 1, etc.
Each file includes entries each whole number of degrees 0 - 360, inclusive.

Note that azimuth angle 0 corresponds to North. 
Source for this information: https://deepspace.jpl.nasa.gov/dsndocs/810-005/301/301K.pdf.
See Figure 11, which lists the cardinal directions under their corresponding azimuth angles on the horizontal axis.

File names with 20kW refer to the 20kW broadcast mask
File names with rcv refer to the receive mask
